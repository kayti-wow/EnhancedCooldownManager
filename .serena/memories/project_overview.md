# Enhanced Cooldown Manager (ECM)

WoW addon displaying customizable resource bars anchored to Blizzard's Cooldown Manager viewers.

**Tech**: Lua 5.1, Ace3 (AceAddon, AceEvent, AceConsole, AceDB, AceConfig), LibSharedMedia-3.0  
**Build**: None - live reload via `/reload`  
**Commands**: `/ecm on|off|toggle|debug`  
**API docs**: https://www.townlong-yak.com/framexml/beta/Blizzard_APIDocumentation
